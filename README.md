 └────────────┬────────────┘             └────────────┬────────────┘
              │                                       │
              │                      ========== [ 핵심 추가 블록 ] ==========
              │                      ▼                                      ▼
              │                  ┌──────────────────────────────────────────────┐
              │                  │ 2. 글로벌 모멘텀 계산 (Global Momentum)      │
              │                  │  ① 파티클 가중 평균: x̂_k = Σ w_i · x_k(i)  │
              │                  │  ② 글로벌 잔차: ε_k = z_k - h(x̂_k)           │
              │                  │  ③ 2차 모멘텀: s_k = β·s_{k-1} + (1-β)·ε_k²│
              │                  └────────────────────┬─────────────────────────┘
              │                                       │
              │                                       ▼
              │                  ┌──────────────────────────────────────────────┐
              │                  │ 3. 관측 노이즈 공분산(R) 팽창 (Inflation)    │
              │                  │  ▶ R_k = R_nom + λ · diag(s_k)             │
              │                  └────────────────────┬─────────────────────────┘
              │                      ========================================
              ▼                                       ▼
 ┌─────────────────────────┐             ┌─────────────────────────┐
